# vainstats
Retrieve game data for Vainglory players using the [Vainglory API](https://developer.vainglorygame.com/docs). 

Get your API key [here](https://developer.vainglorygame.com/). 

# Usage
```python
./get_stats.py -r <region> -u <player name> [-k /path/to/api_key.txt]
```
- Note: After signing up for the Vainglory API, place your API key in the file `key.txt` in the same directory as this script. Or supply the path to the text file containing your key with the `-k` argument.
